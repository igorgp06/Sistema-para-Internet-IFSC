## Sistema de Gerenciamento Imobiliário

Aqui você poderá encontrar o projeto feito para o seminário de Programação Orienteda a Objetos, cujo tema é um sistema de gerenciamento imobiliário.

### Descrição do Projeto

O sistema de gerenciamento imobiliário é uma aplicação desenvolvida em Python que permite a administração eficiente de propriedades imobiliárias. Ele oferece funcionalidades para cadastrar, visualizar, atualizar e excluir informações sobre imóveis, bem como gerenciar clientes e transações.

Todos os conceitos de Programação Orientada a Objetos (POO) foram aplicados no desenvolvimento deste sistema, incluindo encapsulamento, herança, polimorfismo e abstração.
