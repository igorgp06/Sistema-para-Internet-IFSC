## Sistema de Gerenciamento Imobiliário

Aqui você poderá encontrar o projeto feito para o seminário de Programação Orienteda a Objetos, cujo tema é um sistema de gerenciamento imobiliário.

### Descrição do Projeto

O sistema de gerenciamento imobiliário é uma aplicação desenvolvida em Python que permite a administração eficiente de propriedades imobiliárias. Ele oferece funcionalidades para cadastrar, visualizar, atualizar e excluir informações sobre imóveis, bem como gerenciar clientes e transações.

Todos os conceitos de Programação Orientada a Objetos (POO) foram aplicados no desenvolvimento deste sistema, incluindo encapsulamento, herança, polimorfismo e abstração.

Os detalhes podem ser encontrados na pasta [`docs`](./docs/), onde estão disponíveis o pdf de exigência criadas pelo corpo docente.     

### Como Executar o Projeto

Para executar o projeto, siga os passos abaixo:

1. Certifique-se de ter o Python instalado em sua máquina. Você pode baixar a versão mais recente do Python em [python.org](https://www.python.org/downloads/). Para este projeto, recomenda-se o uso do Python 3.8 ou superior.

2. Clone este repositório para o seu computador usando o comando:
   ```bash
   git clone https://github.com/igorgp06/Sistema-para-Internet-IFSC
   ```

3. Navegue até o diretório do projeto raiz do projeto. 
    ```bash
    cd ./Sistema-para-Internet-IFSC/02-2025/POO/seminario-POO-2/
    ```

4. Instale as dependências necessárias utilizando o pip:
   ```bash
   pip install -r requirements.txt
   ```

5. Execute o arquivo principal do sistema:
   ```bash
   python3 -m src.main
   ```
