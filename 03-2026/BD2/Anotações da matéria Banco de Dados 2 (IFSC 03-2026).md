
## 1.0 Informações Gerais

Para entrar no console do `mySql` nós usamos o comando:  
~~~mysql
mysql -u root -p
~~~
com esse comando torna possível acessar o terminal padrão do MySql, tornando assim possível rodar os comandos do próprio. Para ver a lista de comandos básico e as palavras reservadas acesse [hello world.sql](./arqvsSigaa/hello%20world.sql).  

É importante ressaltar que antes de rodar um comando SQL ele deverá conter ao final de cada instancia o ponto e virgula (;) para sinalizar o fim da linha, como no Java por exemplo.
